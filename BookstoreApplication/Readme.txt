The BookstoreApplication is a Http requesting handling server created in Express framework. 

The main purpose of this application is to implement the Conversational State of a web app using the session api provided by Express.

The app handles all the client side Html validation through the servers. (Validtion is done through the backend)

It is a robust app able to handle the different levels of
4xx and 5xx http request errors.
